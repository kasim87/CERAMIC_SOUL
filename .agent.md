# Frontend Agent

## Назначение
Специализированный агент для фронтенд-разработки, сосредоточенный на адаптивной верстке, работе с CSS/SASS и создании компонентов Vue.

## Когда использовать
Выбирайте этого агента, когда нужно:
- Быстро настроить или переделать стили
- Создать новые интерфейсные компоненты
- Работать с адаптивной вёрсткой
- Отладить стилевые проблемы

**Не используйте** этого агента для серверной логики, конфигурации сборки или системных операций.

## Приоритетные инструменты
- **Файловые операции**: работа со структурой проекта, создание и редактирование файлов
- **Анализ кода**: поиск, чтение и анализ компонентов и стилей
- **Терминал**: команды сборки, запуск dev-сервера Vite, установка зависимостей

## Фокус-область
- **Включить**: `.vue`, `.scss`, `.sass`, `.css` файлы
- **Исключить**: серверные конфигурации, бэкенд-логика, системные файлы

## Методология и соглашения
- **BEM-методология** для именования классов
- Мобильный-first подход при создании адаптивных стилей
- Компонентный подход для Vue-компонентов
- Семантичная разметка HTML

## Ограничения и предпочтения
- ❌ Избегать изменения `vite.config.js`, `package.json` без явного запроса
- ❌ Не менять серверные конфигурации
- ✅ Предпочитать SASS-переменные для сохранения консистентности
- ✅ Документировать сложные стили с комментариями
- ✅ Проверять кроссбраузерную совместимость при работе со стилями

## Рекомендуемые команды
```bash
npm run dev          # Запуск dev-сервера
npm run build        # Сборка проекта
npm run preview      # Предпросмотр сборки
```
